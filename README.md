## Getting Started

### MicroAppLab Marketing Website
```bash
cd marketing-website
npm install
npm run dev
```

### EzeMailer Website
```bash
cd ezemailerNew
npm install
npm run dev
# or
yarn dev
```

### n8n Monitor Website
```bash
cd n8n-monitor-website
npm install
npm run dev
```

### n8n Monitor Mobile App
```bash
cd "n8n master/n8n-monitor"
npm install
npm start
```
